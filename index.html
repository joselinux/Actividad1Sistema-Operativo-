¿Qué es un Sistema Operativo?

Es el software que gestiona el hardware y permite ejecutar programas.

Componentes de un Sistema Operativo

Kernel, gestor de procesos, memoria, archivos, controladores, interfaz

Tareas del Sistema Operativo

Administrar recursos, ejecutar programas, gestionar archivos y dispositivos

Funciones de un Sistema Operativo

Controlar procesos, memoria, archivos, dispositivos y seguridad

Características de un Sistema Operativo

Multitarea, multiusuario, estable, seguro,

Tipos de Sistema Operativo

Monousuario, multiusuario, multitarea, tiempo real, distribuido

Sistemas Operativos Para PC

Windows, macOS, Linux

Sistemas Operativos Móviles

Android, iOS, HarmonyOS

Según el Ordenador y Aplicaciones

PC (Windows), servidor (Linux), móvil (Android/iOS)

Compatibilidad de los Sistemas Operativos

Capacidad de ejecutar software y hardware específico o adaptado.

Función básica del Sistema Operativo

Gestionar recursos y facilitar el uso del hardware.

Tres Sistemas Operativos más utilizados

Windows, Android, macOS.

Tareas que realizan los Sistemas Operativos

Ejecutar programas, manejar memoria, controlar dispositivos.

¿Qué es un sistema operativo multitareas?

Permite ejecutar varios programas al mismo tiempo.

¿Qué es un sistema operativo multiusuario?

Permite que varios usuarios lo usen simultáneamente.

¿Cómo funciona un sistema operativo?

Coordina hardware y software desde el arranque del sistema.

Uso en entorno gráfico

Se maneja con ventanas, iconos, menús y ratón.

¿Cuándo inicia el sistema operativo?

Al encender el equipo, después del BIOS/UEFI.

Dos interfaces del Sistema Operativo

Gráfica (GUI) y de línea de comandos (CLI).

¿Qué es un proceso?

Programa en ejecución.

Sistema operativo distribuido y centralizado

Distribuido: varios equipos conectados. Centralizado: todo en un solo servidor.

Sistemas de archivos usados

FAT32, NTFS, ext4, APFS, HFS+.

Otras características actuales

Virtualización, actualizaciones, nube, soporte ARM, seguridad.

¿Qué permite la exigencia de muchos controladores en un sistema operativo?

Permite que el sistema operativo sea compatible con diversos dispositivos de hardware.
